# 🧪 Unit Testing for Frontend & Backend Modules

## 🎯 Objective

To test backend APIs using Pytest and frontend forms using Vitest.

---

## 🗂️ Structure

Testing/

* Backend/
* Frontend/

---

## 🔬 Backend Testing (Pytest)

Run tests:
cd Testing/Backend
pytest

Tested:

* API responses
* Status codes
* Error handling

---

## 🎨 Frontend Testing (Vitest)

Run tests:
cd Testing/Frontend
npm install
npm run test

Tested:

* Form inputs
* Validation
* UI rendering

---

## 🔄 CI Integration

Used GitHub Actions to automatically run tests on push and pull requests.

---

## ✅ Result

* Backend and frontend tested successfully
* Bugs identified and fixed
* Improved reliability

---

## 🚀 Conclusion

Unit testing ensures correctness of application and reduces future errors.
