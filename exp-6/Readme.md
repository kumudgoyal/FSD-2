# Experiment 6: React Forms with Material UI

## Learning Outcomes
1. Install and configure Material UI in React applications
2. Use Material UI form components (TextField, Checkbox, Radio, etc.)
3. Implement form validation with Material UI components
4. Create responsive and styled forms using Material UI
5. Handle form state and submissions with Material UI
6. Customize Material UI component themes and styles
7. Build professional-looking forms with validation feedback

## About This Experiment
This experiment demonstrates building comprehensive forms using Material UI components. It showcases:
- TextField for input fields with validation
- Checkbox and Radio components for selections
- Form validation with error messages
- Responsive form layout using Material UI's Box and Container
- Proper form submission handling
- Integration of Material UI with React state management

The app includes a registration form with multiple input types and validation rules.

## Technologies Used
- React
- Material UI (@mui/material)
- Vite (build tool)
- ESLint (code linting)

## Getting Started
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open your browser to the provided URL

## Project Structure
- `src/components/` - Form components
  - `Form.jsx` - Material UI form with validation and multiple input types
